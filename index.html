Sobre o Projeto

Bem-vindos à Biblioteca Book Love ! Neste projeto individual, fui desafiada a criar um banco de dados para uma biblioteca fictícia, aplicando conceitos de modelagem, criação de tabelas, inserção de registros e criação de relacionamentos onde o contexto é:

"Uma biblioteca está desenvolvendo um sistema web para armazenar informações sobre seus livros e autores e necessita de um Banco de Dados para usar nesse site. Você foi contratado para construir a modelagem do banco de dados!""

Modelagem

Para iniciar o projeto, realizei uma análise cuidadosa dos requisitos solicitados, garantindo que cada entidade e atributo relevante estivesse representado de forma adequada. A modelagem foi feita com base em uma estrutura lógica, buscando facilitar o entendimento e manutenção futura do banco de dados, oferecendo uma abordagem visual e intuitiva para a modelagem de bancos de dados

DBDiagram

O dbdiagram é uma ferramenta poderosa para a modelagem de bancos de dados. Com sua interface amigável e funcionalidades avançadas, é possível criar diagramas de entidade-relacionamento de forma rápida e eficiente. Com o recurso de automática de código SQL e exportação de diagramas, o dbdiagram se torna uma solução valiosa para agilizar o processo de desenvolvimento de bancos de dados, garantindo a consistência e a qualidade do projeto.

Mysql Workbench

Diagrama de relacionamento criado pela engenharia reversa no Workbench revela a estrutura interconectada do banco de dados, exibindo de forma clara e visual as entidades e seus vínculos, sendo possível compreender de maneira intuitiva a complexidade das interações entre as tabelas, facilitando a manutenção e otimização do banco de dados
